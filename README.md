# Portafolio personal — Laura Alejandra González

Este proyecto es un portafolio web personal desarrollado como parte del Sprint 4 del bootcamp de desarrollo web. El sitio presenta mi perfil profesional, habilidades, herramientas, proyectos destacados y formas de contacto.

## Funcionalidad

- Presentación personal.
- Sección de habilidades y herramientas.
- Tarjetas de proyectos con enlaces a demo en vivo y repositorio de GitHub.
- Diseño responsivo para escritorio, tablet y celular.
- Publicación en GitHub Pages.

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- Metodología BEM
- Git y GitHub
- GitHub Pages
- Figma
- Fuentes locales con `@font-face`
